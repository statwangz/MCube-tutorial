Visium human dorsolateral prefrontal cortex dataset
===================================================

The human DLPFC dataset profiled by 10x Visium :cite:`maynard2021transcriptome` is available at
the project website (https://research.libd.org/spatialLIBD/).
The human DLPFC snRNA-seq dataset profiled by 10x Genomics Chromium :cite:`nagy2020single` is available at
the GEO DataSets database with accession number `GSE144136 <https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE144136>`_.
